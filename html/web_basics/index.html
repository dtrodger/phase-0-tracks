This is a paragraph tag. A paragraph. A paragraph tag is used to seperate paragraphs of content. My 4th grade english teach would be upset if I marked this up as a paragraph and did not put at least three sentances in it, so this final sentance is extraneous information. Mr. Scholer was the bomb.


If you wrap a word with the strong tags, the word appears bold. The strong tags also provide semantic meaning by pointing out that certain words in a section are more important than others.


If you wrap a word with the em tags, the word appears bold. THe em tag is similar to the strong tag because it provides semantic meaning that the content between the tags is important. The styling effect of both strong and em can be change or removed with css, but screenreading software would still interpret the word as important.


  • If you wrap content in ul tags,
  • then break the content up and divide it between li tags within the ul tags,
  • the content will be rendered as an unordered list.

  1. If you wrap content in ol tags,
  2. then break up the content and divide it between li tags within the ol tags,
  3. the content will be rendered as an ordered list like this one!
